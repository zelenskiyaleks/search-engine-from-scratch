# Search Engine From Scratch

Building a modern search engine step by step.

## Learning Goals

This project is intended to understand the building blocks of modern search systems:

- Inverted Index
- TF-IDF
- BM25 Retrieval
- Dense Retrieval
- Approximate Nearest Neighbor Search (HNSW)
- Cross-Encoder Reranking
- Retrieval Evaluation Metrics

## Roadmap

- [ ] BM25 Retrieval
- [ ] Dense Retrieval
- [ ] HNSW Index
- [ ] Cross Encoder Reranking
- [ ] Evaluation
